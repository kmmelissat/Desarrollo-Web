꒷꒦︶꒷꒦︶ ๋ ࣭ ꒷꒦꒷꒦︶꒷꒦︶ ๋ ࣭ ⭑꒷꒦

｡ ₊°༺❤︎༻°₊ ｡ Guía 1 ｡ ₊°༺❤︎༻°₊ ｡

En la Guía 1, los ejercicios se enfocan en usar estilos CSS de diferentes formas: desde ponerlos directamente en las etiquetas HTML, pasando por la etiqueta style en el documento, hasta usar un archivo CSS externo para mantener todo más ordenado. Creé varias páginas HTML aplicando colores, tamaños de fuente, alineación y más estilos. Además, como extra, hice una página sobre mi programa favorito y subí los ejemplos a GitHub Pages.

₊˚ʚ ᗢ₊˚✧ ﾟ. Guía 2 ₊˚ʚ ᗢ₊˚✧ ﾟ.

En la Guía 2, los ejercicios suben de nivel con el uso de selectores CSS más complejos y el modelo de caja. El primer ejercicio consiste en estilizar una página HTML usando selectores avanzados, pseudo-clases y pseudo-elementos. El segundo reto es diseñar un tablero de damas aplicando la pseudo-clase :nth-child(). Como extras, creé una biografía personal con estilos más avanzados y experimenté con Google Fonts para personalizar las tipografías.

ଓ༉‧.⭒ֶָ֢⋆. Guía 3 ଓ༉‧.⭒ֶָ֢⋆.

En estos ejercicios, la idea es usar técnicas de posicionamiento y visualización CSS para crear el diseño de un sitio web. El primer ejercicio fue construir una página sobre nominaciones a los Óscares, usando distintos tipos de posicionamiento (relativo, absoluto y flotante) para organizar los elementos en la pantalla. También se usaron propiedades como display, overflow y z-index para controlar cómo se ven los elementos. El segundo ejercicio fue diseñar un menú horizontal multinivel con listas y estilos CSS. Por último, el tercer ejercicio consistió en crear un diseño fijo para una página de una agencia de viajes.

ʚ・ ୨୧・ ɞ Guía 4 ʚ・ ୨୧・ ɞ

Esta guía se enfoca en la creación y estilización de formularios en HTML5, además de implementar diseño responsivo. El primer ejercicio fue construir un formulario alineado sin usar tablas, aprovechando varios tipos de controles HTML5. El segundo ejercicio exploró más controles de formularios, y el tercero se centró en enviar la información. Luego, diseñé un formulario de login con estilos CSS en el cuarto ejercicio. Para el quinto, empecé a usar media queries para hacer un sitio responsivo, y el sexto trató de ajustar el diseño de una agencia de viajes para que se vea bien en diferentes dispositivos.

⋆⑅˚₊ Desafío Práctico ⋆⑅˚₊

El desafío consiste en construir un sitio web para la Bolsa de Trabajo de ESEN usando HTML y CSS, con cuatro páginas: index.html, ofertas.html, aplicar.html y consejos.html. La meta es crear un diseño responsivo siguiendo una imagen de referencia que se nos proporcionó. En la página ofertas.html, hay que presentar una tabla con las ofertas de empleo vigentes. En aplicar.html, se necesita un formulario para que los estudiantes puedan postularse a las ofertas, asegurando que los datos se envíen a aplicar.php sin que aparezcan en la URL. Por último, en consejos.html, hay que insertar un video con una imagen de portada que se muestre hasta que se inicie la reproducción.

⋆ ˚｡⋆୨♡୧⋆ ˚｡⋆ Proyecto: Teatro Nacional ⋆ ˚｡⋆୨♡୧⋆ ˚｡⋆

El proyecto consistió en crear una página web responsiva para el Teatro Nacional, en la que, como grupo, diseñamos una interfaz atractiva utilizando HTML y CSS. La página incluye un menú de navegación intuitivo, secciones interactivas con detalles sobre eventos y opciones de compra de boletos, y una galería de imágenes que resalta momentos destacados de las presentaciones. Además, implementamos un diseño responsivo para asegurar que el sitio se adapte a diferentes dispositivos y un formulario de contacto para consultas. Este proyecto no solo nos permitió aplicar nuestras habilidades técnicas, sino que también fomentó la colaboración y el trabajo en equipo, resultando en un sitio web funcional que refleja la riqueza cultural del Teatro Nacional.
